Smart Cafe Ordering System

Description
A Smart Cafe Ordering System built in C++ using Object-Oriented Programming (OOP) concepts including classes, inheritance, file handling, and exception handling.

Features
- Dynamic Menu Management (Add/Update/Delete items)
- Order System with Quantity Support
- Remove Items from Order
- Billing System (Subtotal + 5% Tax)
- Customer Order History
- File Handling (Save/Load Menu & Orders)
- Search & Filter Items
- Exception Handling for safe execution

OOP Concepts Used
- Classes and Objects
- Inheritance (Person → Customer/Staff)
- Encapsulation
- Vectors for dynamic data

Project Structure
SmartCafe/
- include/
- src/
- data/
- Screenshots/

How to Run
g++ -std=c++17 -o SmartCafe src/*.cpp
./SmartCafe
